### Prerequisites
- [Node.js (>= 18)](https://nodejs.org/en/download)  
- [Git](https://git-scm.com/downloads)  

### Installation

1. Clone the repository  

   ```bash
   git clone https://github.com/Some0ne11/asset-agreements.git
   cd asset-agreement
   ```

2. Install dependencies  

   ```bash
   npm install
   ```

3. Run locally  

   ```bash
   npm run dev
   ```

4. Build for production  

   ```bash
   npm run build
   npm start
   ```
